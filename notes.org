*** keyboard settings
**** in ipad(세팅 - 일반 - 하드웨어 키보드 - 보조키)
     1. Caps Lock -> Control
     2. Control -> Caps Lock
     3. Option -> Command
     4. Command -> Option

**** in blink shell(settings - keyboard)
     1. (Control ->)Capslock : None, Default
     2. Shift : None, Shift
     3. (Capslock ->)Control : None, Control
     4. (Command ->)Option : None, Esc
     5. (Option ->)Command : None, Default
